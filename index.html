GitAhead is a graphical Git client designed to help you understand and manage your history. It features a fast native interface that shows you more of what you want to see, your source code.

GitAhead was designed by SciTools, the makers of Understand. It has all of the features you expect from a commercial quality client, now completely free and open source.

Git provideds a wealth of historical information that, too often, goes largely unexploited because it's too cumbersome to explore with other tools. We believe that when you understand your history you'll make better choices today!

How to Get Help

Ask questions about building or using GitAhead on Stack Overflow by including the 'gitahead' tag. Report bugs by opening an issue in the issue tracker. If you still need help, email us at support@gitahead.com.

How to Contribute

We welcome contributions of all kinds, including bug fixes, new features, documentation and translations. Contribute by following the typical GitHub workflow for pull requests. See the README for more information.
